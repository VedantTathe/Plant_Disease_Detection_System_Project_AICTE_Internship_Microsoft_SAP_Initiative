You can acces this website from folllowing url https://plant-disease-detection-system-vedant-tathe.streamlit.app/ 
If you have to setup project in your laptop then download this repository as zip file and then add dataset and test images/folder in it after that you can run ipynb files step by step.
And If you want to run the website in your local system then open command prompt in given folder and type following commands

1) pip install -r requirements.txt
2) pip install streamlit
3) streamlit run main.py

Now you can upload leafs images and predict plant diseases.
